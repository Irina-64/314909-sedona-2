#### 4.22. Базовая стилизация всех страниц 

1. В форке вашего проекта создайте, положите в папку styles/ и подключите к странице стилевой файл styles.css, чтобы у вас получилась примерно такая структура проекта:

project

| – styles

| –---  styles.css

| – fonts

| – images

| ---- [ваши картинки]

|   index.html

|   catalog.html

2. В форке вашего проекта начните стилизацию страницы, в которую входит:

   - Подключение нестандартных шрифтов в формате WOFF2 со свойством font-display: swap. Шрифтовые файлы лежат у вас в папке fonts/ проекта.

   - Описание текстовых параметров для всех элементов:
      * размер шрифта,
      * высота строки,
      * цвет шрифта,
      * насыщенность шрифта,
      * гарнитура (название шрифта),
      * другие текстовые параметры, если требуется по макету.

Базовые текстовые параметры, необходимые для всего проекта, укажите на <body>. Остальные указывайте точечно на необходимом элементе.

   - Описание фоновых параметров (цвет фона, расположение, размер и повторение фонового изображения). У каждого фона должен быть контрастный цвет текста.

3. Подключите декоративные изображения в стилях:

   - достаточно подключить несколько крупных фоновых изображений или текстур, мелкие иконки можно не подключать.

4. Откройте в браузере свой проект на GitHub и перенесите с помощью перетаскивания все файлы на страницу проекта.

5. Укажите название изменению вашего проекта и нажмите на кнопку Commit message.
