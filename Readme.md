В форке вашего проекта создайте сетку страниц и крупных блоков страниц.

  1.1. Расположите элементы страниц по сетке.
    * Стики футер: min-height, height для <html> и <body>.
    * Крупные блоки и блоки, количество которых не изменяется.
    * Используйте свойства: flex, flex-grow, justify-content.
    * Горизонтальные размеры и отступы: width, max-width, margin, padding.
 1.2. Для удобства можно использовать способ визуального выделения элементов с помощью свойства outline (для удобства также можно использовать разные цвета). Использовать свойство border в качестве такого визуального выделения не стоит, так как border будет влиять на ширину блока при использовании свойства (по умолчанию, для всех элементов на страницах задано box-sizing: content-box).
